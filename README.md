# Amemiya Web

Projeto desenvolvido em **React + Vite + TailwindCSS**.  

---

## Como Usar

1. **Instale o Node.js**  
  [Baixe aqui](https://nodejs.org/pt) a versão recomendada.

2. **Clone este repositório**  
  ```bash
  git clone https://github.com/gfrigo/amemiya-web.git
  cd amemiya-web
  ````

3. **Instale as dependências**  
  ```bash
    npm install
  ````

4. **Execute o projeto**  
  ```bash
    npx vite
  ````
